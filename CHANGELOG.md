### 踩坑记录
| 坑 | 教训 |
|----|------|
| 指针一次性讲完再出题 | 逐概念确认后再推进 |
| typedef 用简化框架讲 | 复杂语法直接用完整声明范式 |
| 地址/偏移用抽象描述 | 硬件概念第一次必须用最强实物类比 |
| 项目代码包进菜单结构 | 大型程序先搭框架再填功能，拆到不能再拆 |
| scanf 输入值忘了 & | 浮点/整数输入要加 &，字符串不要 |
| if(x!=1\|\|2\|\|3) 比较逻辑 | C 不支持链式比较，要用 x!=1 && x!=2 && x!=3 |
| 括号不配对导致 else 找不到 if | 大段代码注意缩进对齐，养成分段编译习惯 |
| 字符串字面量在只读区 | char *s="Hi" 不能改，char s[]="Hi" 可以 |
| NULL vs '\0' 语义混用 | 指针用 NULL，字符用 '\0'，都是 0 但意图不同 |
