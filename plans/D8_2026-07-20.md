### ① 二级指针（**p）

**讲解内容：**
- 从一维指针引申：`int a`, `int *p = &a`, `int **pp = &p`
- 画三层盒子图：a=[5] ← p=[地址] ← pp=[地址]
- 一级指针能改 `*p`（改 a 的值），但不能改 p 本身
- 二级指针能改 `*pp`（改 p 的值）—— 这就是它的实战意义

**英语科普：** pointer = 指针（point=指向 + er=者）。double pointer = 二级指针（double=双重的）

**安全彩蛋：** 二级指针在链表漏洞中常用于构造任意地址写——攻击者通过修改指针的指针来篡改内存布局

**📝 学完立刻出题：**
```c
void set_null(int **pp) {
    *pp = NULL;
}
int x = 42;
int *p = &x;
set_null(&p);
// 问：p 现在是什么？画三层盒子图解释
```
（学生答完后确认 → 进入下一个知识点）

### ② 栈帧（Stack Frame）

**讲解内容：**
- 每个函数调用 = 在栈上分配一块空间（栈帧）
- 栈帧里装：参数、局部变量、返回地址
- 画调用栈图：从 main 开始叠，每调一个函数叠一层
- 函数返回 → 栈帧弹出

**英语科普：** stack = 栈（一叠纸/盘子），frame = 帧/框架。Stack Frame = 栈上的"一格"

**📝 学完立刻出题：**
```
void func2() { int z = 3; }
void func1(int x) { int y = 2; func2(); }
int main() { int a = 1; func1(a); return 0; }
```
画三层栈帧图（main → func1 → func2），标出每个帧里的局部变量

---

### ③ 递归函数（栈帧展开与回退）

**讲解内容：**
- 递归 = 函数调用自己 = 每次调用在一个新栈帧上运行
- 画 factorial(3) 的完整过程：
  - 展开（递推）：fact(3)→fact(2)→fact(1)→fact(0) — 每层压栈
  - 回退（回归）：fact(0)=1→fact(1)=1→fact(2)=2→fact(3)=6 — 逐层弹出
- 必要条件：终止条件（base case）+ 递归公式

**安全彩蛋：** 递归不设终止条件 → Stack Overflow → 本质就是栈溢出攻击利用的原理（通过覆盖返回地址劫持控制流）

**英语科普：** recursion = 递归（re=再次，curs=跑，ion=名词后缀→"反复跑回来"）

**📝 学完立刻出题：**
```
手写 factorial(4) 的递归函数，
画出调用栈从展开到回退的完整过程。
（提示：先写代码，再画图）
```

### ④ 指针与数组等价性（复习）

**讲解内容：**
- `arr[i]` 等价于 `*(arr + i)` — 编译器就是这么翻译的
- `arr` 在大部分表达式中退化为首元素指针
- 例外：`sizeof(arr)` — 只有定义处或数组声明的作用域内才知道完整大小
- 传数组给函数 → 退化为指针 → 丢失大小信息

**关联 D7：** 这就是为什么安全 strcpy 必须额外传 dest_size

**📝 学完立刻出题：**
```c
void func(int arr[]) { printf("%lu", sizeof(arr)); }
int main() {
    int a[100];
    printf("%lu ", sizeof(a));
    func(a);
}
// 输出多少？为什么不一样？
```
