## 已完成

### 1. 数组栈（array_stack.c）
- 结构体：`{int data[MAX_SIZE]; int top;}`
- 空栈：top = -1
- push：先检查满栈，`data[++top] = value`
- pop：先检查空栈，`int val = data[top--]`
- MAX_SIZE 宏定义

### 2. 链栈（linked_stack.c）
- push = 链表头插法：`new->next=*top; *top=new`
- pop = 删头节点：保存data→保存next→free→更新*top
- valgrind 零泄漏 ✅

### 3. 括号匹配（bracket_match.c）
- 左括号 push，右括号用 isMatching(peek, c) 检查栈顶
- 不匹配 → false，匹配 → pop
- 最后栈空才算匹配成功
- 5个测试用例全部正确 ✅
