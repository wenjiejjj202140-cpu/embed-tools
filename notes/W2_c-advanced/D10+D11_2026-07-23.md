## 一、va_list 可变参数

### 四步法
```c
#include <stdarg.h>

int sum(int count, ...) {
    va_list ap;            // ① 声明
    va_start(ap, count);   // ② 初始化（告诉它从 count 后面开始）

    int total = 0;
    for (int i = 0; i < count; i++)
        total += va_arg(ap, int);  // ③ 逐个取出（每次都要指定类型）

    va_end(ap);            // ④ 清理
    return total;
}
```

### 关键点
- `va_start(ap, last_named)` — 需要最后一个命名参数作为锚点
- `va_arg(ap, type)` — 类型由调用者指定，不匹配就是未定义行为
- char/short 在变参中自动提升为 int（`va_arg(ap, char)` 实际上是取 int）

## 二、union 联合体

### 核心概念
- 所有成员**共享同一块内存**
- `sizeof(union)` = 最大成员的大小
- 写一个成员会覆盖其他成员

### vs struct
| struct | union |
|--------|-------|
| 各住各的（加法） | 挤在一起（最大值） |
| sizeof = 各成员 + padding | sizeof = 最大成员 |

### 嵌入式用途
```c
union Reg32 {
    uint32_t value;         // 整体 32 位
    uint16_t half[2];       // 拆两个 16 位
    uint8_t  bytes[4];      // 拆四个 8 位
};
```

### 大小端
- **小端（x86）：** 低地址存低字节 → union 的 c 读到的是最低字节
- **大端（网络协议）：** 低地址存高字节

## 三、enum 枚举

### 核心
- 给整数起名字，代替魔数
- 默认从 0 递增
- 手动指定值后，后面的基于它递增

```c
enum Color { RED, BLUE=5, GREEN, YELLOW };
// RED=0, BLUE=5, GREEN=6, YELLOW=7
```

## 四、位域 bit field

### 语法
```c
struct Packed {
    unsigned int a : 3;   // 占 3 位
    unsigned int b : 5;   // 占 5 位
    unsigned int c : 8;   // 占 8 位
};
```

### 注意
- N 位无符号范围：0 ~ 2^N - 1
- 溢出截断（只保留低 N 位）
- 有符号位域：最高位为符号位，范围减半
- 位域顺序由编译器决定，不跨平台

## 五、单向链表

### 节点定义
```c
struct Node {
    int data;
    struct Node *next;  // 指向下一个节点
};
```

### 链表 vs 数组
| | 数组 | 链表 |
|--|------|------|
| 内存 | 连续 | 离散 |
| 大小 | 固定 | 动态增长 |
| 随机访问 | O(1) | O(n) |
| 插入/删除 | O(n) | O(1) |
| 每节点额外 | 无 | 一个指针 |

### 头插法
```c
void insert_head(struct Node **head, int value) {
    struct Node *new = malloc(sizeof(struct Node));
    new->data = value;
    new->next = *head;  // 新节点指向旧头
    *head = new;        // 更新头指针
}
```
传二级指针的原因：要修改外部的 head 本身。

### 按值删除
```c
int delete_by_value(struct Node **head, int value) {
    struct Node *prev = NULL, *cur = *head;
    while (cur) {
        if (cur->data == value) {
            if (prev == NULL)         // 删头节点
                *head = cur->next;
            else                      // 删其他节点
                prev->next = cur->next;
            free(cur);
            return 1;
        }
        prev = cur;
        cur = cur->next;
    }
    return 0;
}
```
