# D1 嵌入式初识与点灯 — 2026/7/12

## 嵌入式系统是什么

- **PC vs 开发板：** 开发板 = 缩水版电脑，但所有外设都直接暴露给程序员
- **架构：** CPU + ROM（存程序） + RAM（存数据） + 外设（GPIO/UART等）
- **MCU = Microcontroller Unit**（微控制器），片上集成 CPU/ROM/RAM/外设

## GPIO 基础

- **GPIO** = General Purpose Input Output（通用输入输出引脚）
- **方向：** 输入（读电平） / 输出（写电平）
- **电平：** 高电平（3.3V/5V ≈ 1）/ 低电平（0V ≈ 0）

```c
// HAL 库点灯伪代码
HAL_GPIO_WritePin(LED_GPIO_Port, LED_Pin, GPIO_PIN_SET);  // 点亮
HAL_GPIO_WritePin(LED_GPIO_Port, LED_Pin, GPIO_PIN_RESET); // 熄灭
```

## 英语科普

| 词汇 | 含义 | 记忆 |
|------|------|------|
| register | 寄存器 | 拉丁语 regesta（记录列表）→ CPU 里的小本本 |
| volatile | 易变的 | 告诉编译器"别优化我，我的值随时会变" |
| mask | 掩码 | 像面具一样过滤位 |

## 实操：LED 闪烁模拟

```c
#include <stdio.h>
#include <unistd.h>  // usleep

int main() {
    while (1) {
        printf("LED ON\n");
        usleep(500000);  // 500ms
        printf("LED OFF\n");
        usleep(500000);
    }
    return 0;
}
```

## 编程规范第一次提醒

- `void`：明确表示"不接收参数"，比空括号规范
- `#include <unistd.h>`：usleep 的声明头文件
