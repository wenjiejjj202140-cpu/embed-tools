# D3 指针基础 — 2026/7/13

## 指针的本质

> **指针 = 地址**（内存单元的编号）

```c
int a = 5;
int *p = &a;   // p 存的是 a 的地址，不是 a 本身
```

## 两个关键符号

| 符号 | 名字 | 作用 |
|------|------|------|
| `&` | 取地址符 | 拿到变量的地址（编号） |
| `*` | 解引用符 | 通过地址找到那个值 |

```c
int a = 10;
int *p = &a;

printf("%p\n", p);  // 地址值（如 0x7fff...）
printf("%d\n", *p); // 10（通过地址找到 a 的值）
```

## 传值 vs 传址（核心理解 ⭐）

```c
// ❌ 传值 — 函数拿到的是复印件，改不了原件
void swap(int x, int y) {
    int tmp = x;
    x = y;
    y = tmp;  // 改的是复印件，外面的 a,b 不变
}

// ✅ 传址 — 函数拿到原件地址，通过地址直接改
void swap(int *x, int *y) {
    int tmp = *x;
    *x = *y;
    *y = tmp;  // 原件被改了
}

int main() {
    int a = 5, b = 10;
    swap(&a, &b);  // 传地址过去
}
```

**记忆法：** 你让快递员帮你改桌子上的文件
- 传值 → 你把文件复印了一份寄过去，快递员改的是复印件
- 传址 → 你把桌子的地址告诉快递员，他直接去桌子上改原件

## 指针类型转换

```c
*(unsigned int*)0x40020C14 = 1;
```

**拆解：**
1. `0x40020C14` = 一个数字（寄存器的地址）
2. `(unsigned int*)` = 强制转换成指向 unsigned int 的指针
3. `*` = 解引用
4. `= 1` = 写入 1

> ⏰ 这是 D2 的寄存器操作，现在你应该能彻底看懂了

