# D4 结构体基础 — 2026/7/14

## 结构体是什么
- 把多个不同类型的变量打包成一个整体
- 语法：`struct 标签 { 成员列表 };`

```c
// 定义结构体类型
typedef struct {
    char name[32];
    int age;
    float score;
} Student;
```

- `typedef` = 取别名，这样不用每次写 `struct Student`

## 结构体变量

```c
// 初始化（必须在定义时）
Student s1 = {"文", 20, 85.5};

// 访问成员
s1.age = 21;
```

## 结构体数组

```c
Student class[3] = {
    {"文",   20, 85.5},
    {"小杰", 22, 92.0},
    {"小明", 21, 78.0}
};
```

## 结构体传参（重点 ⭐）

| 方式 | 语法 | 特点 |
|------|------|------|
| 传值 | `void print(Student s)` | 复制整个结构体（40字节），函数内修改不影响原数据 |
| 传指针 | `void print(const Student *s)` | 只传地址（4-8字节），`->` 访问成员，可改原数据 |

```c
// 传值：复印一份，40字节全部压栈 ❌ 嵌入式浪费
void print_student(Student s) {
    printf("%s %d岁 %.1f分\n", s.name, s.age, s.score);
}

// 传指针：只传地址 ✅ 推荐
void print_student(const Student *s) {
    printf("%s %d岁 %.1f分\n", s->name, s->age, s->score);
}
```

## 温故知新：传值 vs 传址

> D3 教 `swap(int *a, int *b)` 时学的，同样是"传指针才能改原数据"
